# JavaScript-Game-1
This is a point and shoot game that I learned how to develop from the
JavaScript Game Development For Beginners Course on Youtube.com.

Features:

1.) Point with cursor to aim at ravens.
2.) Left click to shoot.
3.) Restart by refreshing the page.

Technology For Development:

1.) HTML
2.) CSS
3.) JavaScript
4.) Visual Studio Code

NO FRAMEWORKS!!!!
